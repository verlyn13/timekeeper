Adaptive Module¶

Adaptive Temporal System (Definition 21)¶

Implementation of the dynamic adaptability components of the AgentTemporal framework.

This module extends the core AgentTemporal class with adaptability features that allow the system to adjust its temporal units based on observed usage patterns.

class AdaptiveAgentTemporal(unit_config=None, agent_count=1)[source]¶

Bases: AgentTemporal

Extension of AgentTemporal that implements dynamic adaptability as described in Section 5 of the paper, in particular:

  • Adaptive Subdivision Function Γ(O, A) (Definition 21)

  • Partition Reconfiguration (Axiom 4)

  • Optimal Temporal Granularity (Property 1)

Initialize the adaptive temporal system.

Parameters:

  • unit_config – Optional custom configuration of time units

  • agent_count – Number of agents in the system (default: 1)
