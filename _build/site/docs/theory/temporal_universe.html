Temporal Universe

Definition 1: Temporal Universe

A temporal universe is defined as a pair $(T, <)$ where:

  • $T$ is a set of timepoints
  • $<$ is a strict total ordering on $T$

This ordering satisfies the following properties for all $t_1, t_2, t_3 \in T$:

  • Irreflexivity: $\neg(t_1 < t_1)$
  • Transitivity: If $t_1 < t_2$ and $t_2 < t_3$, then $t_1 < t_3$
  • Trichotomy: Exactly one of the following holds: $t_1 < t_2$, $t_2 < t_1$, or $t_1 = t_2$

Mathematical Foundation

The temporal universe forms the foundational structure for all temporal operations in the framework. It provides a linear, ordered representation of time that serves as the basis for hierarchical partitioning and temporal operations.

Key Properties

  • Linearity: Time is represented as a linear sequence, maintaining a strict ordering between any two distinct timepoints.
  • Continuity: The temporal universe can be viewed as continuous, though our representation focuses on discrete timepoints.
  • Foundational Role: All other temporal constructs (partitions, operations, morphisms) are built upon this base structure.

Related Definitions

  • Hierarchical Partition (Definition 2): Partitions the temporal universe into a hierarchical structure.
  • Timepoint Operations (Definitions 6-8): Operations defined on timepoints within the temporal universe.

Usage in the Framework

The temporal universe concept is used throughout the framework in the following ways:

  • As the foundation for all temporal operations
  • As the domain for hierarchical temporal partitions
  • As the base for defining morphisms between different time domains
  • As the underlying structure for the scheduler
